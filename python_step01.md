##Python Step‑01 Guide

1. Install Python

Visit the official Python website.

Download the latest stable version.

During installation, enable the option Add Python to PATH.

2. Install a Code Editor

Install Visual Studio Code (VS Code).

Open VS Code and install the Python extension.

3. Verify Python Installation

Open your terminal or command prompt and run:

python --version

You should see the installed version number.

4. Create Your First Python File

Open VS Code.

Create a new file named hello.py.

Add this code:

print("Hello, World!")

5. Run Your First Program

In the VS Code terminal, run:

python hello.py

Expected output:

Hello, World!

6. Step‑01 Complete

You now have Python installed, a code editor ready, and your first program running. You're ready for Step‑02.
